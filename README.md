# PRODIGY_GA_04 – Image-to-Image Translation using Conditional GAN (cGAN)

## Introduction

As part of my Generative AI Internship at Prodigy InfoTech, I developed an Image-to-Image Translation project that demonstrates how Artificial Intelligence can transform one image into another visual representation.

Image-to-Image Translation is a fascinating application of Generative AI that enables the conversion of images from one domain to another while preserving important visual information. This project helped me understand the fundamentals of image transformation, computer vision, and generative modeling techniques.

---

## Project Objective

The primary objective of this project is to transform an input image into a different visual style or representation using image translation techniques inspired by Conditional Generative Adversarial Networks (cGANs).

The project showcases how AI can learn relationships between input and output images and generate meaningful visual transformations.

## Workflow

1. Install and configure the required libraries.
2. Upload an input image.
3. Process the image using image transformation techniques.
4. Generate the translated image.
5. Save and visualize the output.
6. Compare input and translated images.

## Input Image

The project accepts an image as input and performs visual transformation to generate a modified output image.

Examples:

- Sketch to Colored Image
- Black & White to Colorized Image
- Image Style Transformation
- Artistic Image Conversion

## Output

The system successfully generates a transformed image while preserving the key visual characteristics of the original image.

The generated output demonstrates the potential of AI-powered image translation and visual content generation.
